Build docker namenode: 
~$ sudo docker-compose up -d namenode

Turn on services 4 data nodes: 
~$ docker-compose up -d

Type: http://localhost:9870 to check status.

Insert data in Data Lake

Create folder in HDFS: 
~$ docker exec namenode hdfs dfs -mkdir -p /user/data

Put file from folder data-input to HDFS:
~$ docker exec namenode hdfs dfs -put /data-input/drug200.csv /user/data/

Check result overall:
~$ sudo docker exec namenode hdfs dfs -ls /user/data

Check distribution:
~$ docker exec namenode hdfs dfs -ls -R /user/data/
~$ docker exec namenode hdfs fsck /user/data/drug200.csv -files -blocks -locations

If conflict or exception, stop by cli: ~$ sudo docker-compose down and start over.

Update hdfs config: docker-compose up -d
